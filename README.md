# wi-md

Work instructions for converting markdown documents to PDF using typst-based tooling.

## Quick Start

```bash
md-pdf input.md -o output.pdf --template simple
```

## Work Instructions

- [MD-PDF Tool](md-pdf-tool/SKILL.md) — Convert markdown to PDF using `md-pdf` (typst engine)
- [templates/](md-pdf-tool/templates/) — Custom typst templates (includes a sample brand palette)
